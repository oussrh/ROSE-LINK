ROSE Link APT Repository

Official APT repository for ROSE Link - Transform your Raspberry Pi into a VPN router with ad blocking.

Quick Install

curl -sSL https://oussrh.github.io/ROSE-LINK/install.sh | sudo bash

Manual Setup

# Add GPG key
curl -fsSL https://oussrh.github.io/ROSE-LINK/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/rose-link.gpg

# Add repository
echo "deb [arch=arm64,armhf signed-by=/usr/share/keyrings/rose-link.gpg] https://oussrh.github.io/ROSE-LINK stable main" | sudo tee /etc/apt/sources.list.d/rose-link.list

# Install
sudo apt update
sudo apt install rose-link

Supported Architectures

  • arm64 - Raspberry Pi 3/4/5 (64-bit OS)
  • armhf - Raspberry Pi 3/4/Zero 2W (32-bit OS)

Supported OS

  • Raspberry Pi OS (Bullseye/Bookworm)
  • Debian 11 (Bullseye)
  • Debian 12 (Bookworm)
Note: This repository is designed for Raspberry Pi devices. For other platforms, please install from source.
