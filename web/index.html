Informations

Version: ROSE Link Pro v0.1.0

Plateforme: Raspberry Pi 4

Interface WAN: eth0 (RJ45) / wlan0 (WiFi)

Interface AP: wlan1 (Hotspot)

VPN: WireGuard (wg0)

Documentation

  • Connectez-vous au réseau ROSE Link via le hotspot
  • Configurez votre connexion WAN (RJ45 prioritaire, WiFi en fallback)
  • Importez votre profil WireGuard depuis la Fritz!Box
  • Tout le trafic passe par le VPN vers la Belgique
  • Kill-switch actif : pas de fuite si le VPN tombe
